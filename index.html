Truth Table Generator

Instructions

Variables must be either one letter or one letter followed by numbers.

Literal true can be "⊤", "true", or "TRUE".

Literal true can be "⊥", "false", or "FALSE".

Disjunctive operator (or) can be "|", "v", "∨", "or", or "OR".

Conjunctive operator (and) can be "&", "^", "∧", "and", or "AND".

Negation operator (not) can be "!", "~", "¬", "not", or "NOT".

Conditional operator (if) can be "->", "→", "if", or "IF".

Biconditional operator (if and only if) can be "<->", "↔", "iff", or "IFF".
